# 贡献者行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验水平、教育程度、社会地位、国籍、相貌、种族、宗教信仰、性取向，我们都将以尊重和友好的态度对待项目社区中的每一个人。

## 我们的准则

有助于创造积极环境的行为包括但不限于：

* 使用友好和包容性语言
* 尊重不同的观点和经历
* 耐心接受有益批评
* 关注对社区最有利的事情
* 友善对待其他社区成员

不可接受的行为包括但不限于：

* 使用与性有关的言语或图像，以及不受欢迎的性关注或骚扰
* 挑衅、侮辱或贬损的评论，以及人身攻击或政治攻击
* 公开或私下的骚扰行为
* 未经许可地发布他人的隐私信息，例如住址或电子邮箱
* 其他可以被合理地认定为不恰当或者违反职业操守的行为

## 我们的责任

项目维护者有责任为「可接受的行为」标准做出诠释，并对已发生的不当行为采取恰当且公平的纠正措施。

项目维护者有权利和责任去删除、编辑、拒绝违背本行为准则的评论、提交、代码、wiki 编辑、问题等贡献内容，并可暂时或永久地封禁任何他们认为行为不当、威胁、冒犯、有害的贡献者。

## 适用范围

本行为准则适用于所有项目平台，以及当个人代表项目或其社区时的公开场合。代表项目或社区的情形包括但不限于：使用项目官方电子邮件、通过官方社交媒体账号发布消息、在线或线下活动中作为指定代表。

## 监督

对于辱骂、骚扰或其他不可接受的行为，可以通过以下方式向项目团队报告：

**报告渠道**:
- 电子邮件: conduct@colobot.ai
- GitHub Issues: 使用 "行为准则" 标签

所有投诉都将得到及时、公正的审查和调查。项目团队有义务对事件报告者保密。

## 处理流程

### 1. 报告
当您遇到不可接受的行为时，请：
1. 收集相关信息（时间、地点、相关人员、具体行为）
2. 通过上述渠道报告
3. 提供您希望的联系方式

### 2. 调查
项目团队将：
1. 在24小时内确认收到报告
2. 进行初步调查（通常需要1-3个工作日）
3. 必要时联系相关方了解情况
4. 保护报告者的隐私

### 3. 处理
根据调查结果，可能采取的措施包括：
1. **纠正**: 要求停止不当行为并道歉
2. **警告**: 正式警告并记录在案
3. **暂时封禁**: 暂时禁止参与项目活动
4. **永久封禁**: 永久禁止参与项目活动

### 4. 上诉
对处理决定有异议，可在7天内提出上诉。

## 执行

项目维护者如果不致力于或不遵循本行为准则，可能被临时或永久地剥夺作为项目维护者的资格。

## 贡献者权利

作为贡献者，您有权：
1. 在尊重他人的环境中工作
2. 受到公平对待
3. 报告不当行为而不必担心报复
4. 参与决策过程（如适用）

## 维护者指南

### 积极行为
- 热情欢迎新贡献者
- 耐心解答问题
- 提供建设性反馈
- 认可和感谢贡献

### 处理冲突
1. **保持冷静**: 避免情绪化回应
2. **私下沟通**: 敏感问题私下讨论
3. **聚焦问题**: 讨论具体问题，不针对个人
4. **寻求共识**: 寻找双方都能接受的解决方案
